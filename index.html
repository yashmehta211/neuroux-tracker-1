Connect Otter via Zapier
After every call, Otter sends the summary to Zapier which forwards it here. The tracker reads the lead's name from the summary text and auto-assigns it.
Your Webhook URL
https://neuroux-tracker-1.vercel.app/api/otter
Zapier Setup Steps
1. Go to zapier.com → Create Zap
2. Trigger: Otter → New Recording
3. Action: Webhooks by Zapier → POST
4. URL: paste the webhook above
5. Payload type: JSON
6. Fields: title, summary, action_items, created_at
7. Turn on your Zap ✓
