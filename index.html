Otter via Zapier
After each call, Otter sends the summary to Zapier which forwards it here. The tracker extracts the lead name from the summary text and auto-assigns it.
Webhook URL
https://neuroux-tracker-1.vercel.app/api/otter
Zapier Setup
1. zapier.com → Create Zap
2. Trigger: Otter → New Recording
3. Action: Webhooks by Zapier → POST
4. URL: paste webhook above
5. Payload type: JSON
6. Fields: title, summary, action_items, created_at
7. Turn on Zap ✓
